Калькулятор вязкости на показаниях вискозиметра ВЗ-246.

Приложение для определения вязкости испытуемых жидкостей, с помощью вискозиметра ВЗ-246.

Порядок проведения измерений:

1. Выберите диаметр сопла
2. Введите температуру, при которой производятся измерения
3. Рассчитайте время истечения испытуемой жидкости.
4. Калькулятор выдаст вам полученное значение. Для точного расчета проведите измерения 3 раза.
